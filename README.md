# Wind Market Wallboard Project

This directory hosts the implementation for the Wind Market Wallboard described in the project documentation (`docs/`).

Structure:
- `backend/`: Python services (FastAPI gateway, data ingestion, providers).
- `frontend/`: HTML/CSS/JavaScript kiosk client sources.
- `config/`: Edition configuration templates, session schedules, refresh intervals.
- `deployment/`: Docker, systemd, kiosk scripts.
- `infra/`: Infrastructure-as-code, monitoring manifests.
- `tests/`: Shared test assets, integration fixtures.
- `docs/`: Project-specific design notes generated during development.

Refer to `todo.md` in repository root for the active backlog.
